◆ RawMode

enum ls::FileData::RawMode

RAW plain-text mode.

Enumerator
RAW_MODE_DEFAULT 

Default.

Write default. If no direction is in the lower hemisphere, write as hemispherical RAWBH10A. Otherwise, write as spherical RAWBS10A.

RAW_MODE_ONLY_BRDF 

Only BRDF.

Write only BRDF directions in the upper hemisphere as hemispherical RAWBH10A.

RAW_MODE_ONLY_BTDF_AS_BRDF 

Only BTDF, flipped as BRDF.

Write only BTDF directions in the lower hemisphere, flipped to the upper hemisphere, as hemispherical RAWBH10A.
