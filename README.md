# Layered-SQT

Layered assembly to SQT-format BSDF.

### Building from source

```
$ git clone --recursive https://github.com/mgradysaunders/layered-sqt
$ cd layered-sqt
$ mkdir bin && cd bin
$ cmake ..
$ cmake --build .
```
